    Install

    Install via NPM: 
    npm install zmdc
    You can use this library as JavaScript (ESM) or Typescript. If you need UMD format, you can compile the TypeScript files by yourself, using your toolchain.

    ESM

    import {parseExampleFunction, showExampleCode} from 'node_module/zmdc/dist/zmdc.js'

    TypeScript

    import {parseExampleFunction, showExampleCode} from 'node_module/zmdc/lib/zmdc.(t|j)s'
    The extensions depends on your toolchain!

    If you have to rename the function formatFancy to formatColorful, you have to change your example code, too. But you don't want to also change the HTML, where you placed the example code. This library should do it for you.

    Step 3. Mark necessary HTML tags in demonstration function

    Before you can let this library show you demo code on DOM, you have to mark these information in your function:

    1. In which DOM element should the demo code be shown
    2. Which DOM elements needs the demo code to work properly

    For the 1. point: the first line in demo function must be a comment // tag: id The id is the attribute id of the element tag, where JavaScript demo code is placed. Here ist an element with id="fancy-demo". The element does not have to be a div, in most cases a div is fine.

    The given element must contain at least two code-Tags with class list containing example-javascript and example-html respectively.

    For the 2. point: One comment in form // <tag> for each necessary DOM-element. The demo function here just needs only an element with id="ff-result", so there is only one comment. By this way you can directly denote the demo code, which DOM element it needs to work.
